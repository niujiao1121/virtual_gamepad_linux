# 项目要求与说明

## 目标

编写一个运行在 Linux 桌面的虚拟手柄控制面板，通过 `uinput` 创建虚拟 Xbox 风格手柄。

## 版本管理

- 项目必须维护 `VERSION.md`
- 每次发布前同步更新当前版本号与变更记录
- 代码与说明中的版本信息应保持一致

## 环境

- 编译环境：Linux 桌面
- 构建工具：`cmake`、`g++`
- UI：`SDL2` + `Dear ImGui`

## 要求

1. 虚拟手柄应暴露常见 Xbox 手柄按键与轴。
2. UI 布局应尽量接近真实手柄。
3. `LS/RS` 应与摇杆拖拽逻辑分离。
4. 代码需提供中文注释，便于维护。
5. 若有不清晰的地方，优先直接确认，不要含糊实现。
